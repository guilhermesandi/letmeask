## Como usar

```
1- Clone esse repositório

2- Vá no diretório raiz do repositório

3- Instale as dependências:
$ yarn install

4- Inicie o backend:
$ yarn server

5- Inicie a aplicação:
$ yarn dev

6- No navegador:
http://localhost:3000/
```
